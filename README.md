# Stray-animal-adoption-system

流浪动物领养系统

## 项目文件结构

```
Stray-animal-adoption-system/
├── backed/                     # 后端代码目录
├── fronted/                    # 前端代码目录
├── Documents/                  # 项目文档目录
│   ├── Proposal/              # 项目提案文档
│   ├── Requirements/          # 需求文档
│   ├── UI/                    # UI设计文档
│   ├── activity-diagram/      # 活动图
│   ├── flow-chart/            # 流程图
│   └── use-case-diagram/      # 用例图
└── README.md                   # 项目说明文件
```

### 目录说明

- **backed/**: 存放后端服务代码
- **fronted/**: 存放前端应用代码
- **Documents/**: 存放项目相关文档
  - **Proposal/**: 项目提案和计划文档
  - **Requirements/**: 功能需求和技术需求文档
  - **UI/**: 用户界面设计稿和原型
  - **activity-diagram/**: UML活动图
  - **flow-chart/**: 业务流程图
  - **use-case-diagram/**: UML用例图
