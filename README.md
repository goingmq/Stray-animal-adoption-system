# Stray-animal-adoption-system

流浪动物领养系统 - 一个网页开发项目

## 项目简介

本项目是一个流浪动物领养系统的网页应用，旨在帮助流浪动物找到新家。

## 项目结构

```
Stray-animal-adoption-system/
├── frontend/           # 前端代码存放处
├── backend/            # 后端代码存放处
├── docs/               # 文档存放处
│   ├── requirements/   # 成员需求文档（长期更新）
│   └── ui-snapshots/   # 网页UI快照
└── README.md           # 项目说明文档
```

## 目录说明

### frontend/ - 前端代码
存放前端项目的源代码，包括HTML、CSS、JavaScript等文件。

### backend/ - 后端代码
存放后端项目的源代码，包括服务器端逻辑、API接口等。

### docs/ - 文档存放处
- **requirements/** - 针对成员的需求文档存放位置（长期更新）
- **ui-snapshots/** - 网页UI快照存放位置
